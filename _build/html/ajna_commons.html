Remove espaços à direita e esquerda, passa para “casefold”(caixa baixa), usa função normalização para retirar marcas de diacríticos (acentos e

class ajna_commons.flask.log.CustomAdapter(logger, extra)[source]¶

This example adapter expects the passed in dict-like object to have a ‘connid’ key, whose value in brackets is prepended to the log message.

process(msg, kwargs)[source]¶

Process the logging message and keyword arguments passed in to a logging call to insert contextual information. You can either manipulate the message itself, the keyword args or both. Return the message and kwargs modified (or not) to suit your needs.

Normally, you’ll only need to override this one method in a LoggerAdapter subclass for your specific needs.
